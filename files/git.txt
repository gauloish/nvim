Init Repository:
	git init <directory>

Add Changes:
	git add <file>

Commit Changes:
	git commit -m "<comment>"

Rename (Move) Branch:
	git branch --move <from> <to>

Add Branch:
	git branch <branch>

Remove Branch:
	git branch --delete <branch>

Go to Branch:
	git checkout <branch>

Push Local Repo to Remote Repo:
	git remote add origin <url>
